MintBridge XFCE v1.0 — Proyecto técnico comunitario

Descripción:
MintBridge es una interfaz HTML diseñada para entornos XFCE en Linux Mint, con integración visual de recursos gráficos, estructura validada y documentación reproducible.

Contenido del paquete:
- MintBridge.html
- assets/
- checklist-validacion.txt
- version-log.txt
- README.txt
- instalar.sh (opcional)

Requisitos:
- Linux Mint XFCE
- Navegador compatible con HTML local
